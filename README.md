# Sorry Papri - Apology Website 🥺🌸

পাপড়িকে সারপ্রাইজ দেওয়ার জন্য একটি অত্যন্ত সুন্দর এবং ইন্টারঅ্যাক্টিভ ওয়েবসাইট। এখানে আছে একটি সুন্দর অ্যানিমেটেড খাম (envelope), টাইপিং এফেক্টসহ চিঠি, এবং একটি মজার নো (No) বাটন যা মাউস বা টাচ সরালেই দূরে চলে যায়! যখন সে "হ্যাঁ" বাটনে চাপ দেবে, তখন স্ক্রিনজুড়ে চমৎকার কনফেটি বৃষ্টির সাথে আপনার সুন্দর ছবি ও মেসেজ আসবে।

---

## 🚀 সম্পূর্ণ ফ্রিতে হোস্ট করার উপায় (How to Host for Free)

টাকা ছাড়া এই ওয়েবসাইটটি ইন্টারনেটে পাবলিশ করে পাপড়ির সাথে লিংক শেয়ার করার ৩টি সবচেয়ে সহজ উপায় নিচে দেওয়া হলো:

### উপায় ১: Netlify (সবচেয়ে সহজ ও দ্রুত - কোনো কোডিং জ্ঞান লাগবে না)
১. **[Netlify](https://www.netlify.com/)** ওয়েবসাইটে যান এবং একটি ফ্রি অ্যাকাউন্ট খুলুন।
২. লগইন করার পর **"Sites"** অপশনে যান।
৩. একদম নিচে একটি বক্স পাবেন যেখানে লেখা থাকবে: **"Drag and drop your site folder here"**।
৪. আপনার কম্পিউটারের `sorry-papri` ফোল্ডারটি মাউস দিয়ে ড্র্যাগ করে ওই বক্সের উপর ছেড়ে দিন।
৫. মাত্র ৫ সেকেন্ডের মধ্যে আপনার সাইট লাইভ হয়ে যাবে!
৬. Netlify আপনাকে একটি ফ্রি লিংক দিবে (যেমন: `https://something.netlify.app`)। আপনি চাইলে Site Settings থেকে নাম পরিবর্তন করে `https://sorry-papri.netlify.app` বা আপনার পছন্দমতো নাম দিতে পারেন।

### উপায় ২: GitHub Pages (স্থায়ী এবং প্রফেশনাল)
১. **[GitHub](https://github.com/)**-এ লগইন করুন এবং একটি নতুন Repository তৈরি করুন (নাম দিন `sorry-papri`)।
২. এই ফোল্ডারের ফাইলগুলো (`index.html`, `style.css`, `script.js` এবং `cute_sorry_bears.png`) সেখানে আপলোড করুন।
৩. Repository-র **Settings** ট্যাবে যান।
৪. বাম পাশের মেনু থেকে **Pages** অপশনে ক্লিক করুন।
৫. **Build and deployment** সেকশনে Branch হিসেবে `main` বা `master` সিলেক্ট করে **Save** বাটনে ক্লিক করুন।
৬. ১-২ মিনিট পর আপনার ওয়েবসাইটটি লাইভ হয়ে যাবে এবং একটি লিংক পাবেন (যেমন: `https://yourusername.github.io/sorry-papri/`)।

## 🛠️ ফোল্ডার স্ট্রাকচার (Folder Structure)
- `index.html` - ওয়েবসাইটের মূল গঠন।
- `style.css` - চমৎকার ডিজাইন ও অ্যানিমেশন।
- `script.js` - টাইপিং এফেক্ট, নো বাটনের ম্যাজিক এবং কনফেটি এফেক্ট।
- `cute_sorry_bears.png` - সাকসেস স্ক্রিনের কিউট ইলাস্ট্রেশন (AI দ্বারা তৈরি)।
